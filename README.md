## Configuration:
- `USER=` username
- `POSTKEY=` postkey
- `INTERVAL=0` time interval per battle
- `HEADLESS=True` enable/disable headless mode
- `CUSTOM_PRIORITY=False` force custom lineup on certain match
- `KEY=` license key
